## Get started

<Steps>
  <Step title="Sign up for GitHub Copilot">
    Go to [github.com/settings/copilot](https://github.com/settings/copilot) and activate a Free, Pro, or Business plan.
  </Step>
  <Step title="Install the extension">
    Open VS Code and install the **GitHub Copilot Chat** extension from the Extensions Marketplace (search `GitHub.copilot-chat`).
  </Step>
  <Step title="Sign in">
    VS Code will prompt you to sign in with your GitHub account. Approve the authorization in your browser.
  </Step>
  <Step title="Start chatting">
    Open the Copilot Chat panel with `Ctrl+Alt+I` / `Cmd+Option+I`, or press `Ctrl+I` / `Cmd+I` in the editor for inline chat.
  </Step>
</Steps>

<Note>
  GitHub Copilot Free gives you 2,000 code completions and 50 chat messages per month at no cost. Upgrade to a paid plan for unlimited usage. See the [installation guide](/installation) for all subscription options.
</Note>
