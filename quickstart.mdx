---
title: "Quick start"
description: "Get GitHub Copilot Chat running in VS Code in under 5 minutes."
---

Follow these steps to install GitHub Copilot Chat, sign in with your GitHub account, and send your first AI chat message.

<Steps>
  <Step title="Sign up for GitHub Copilot">
    Visit [github.com/settings/copilot](https://github.com/settings/copilot) and activate a plan. The **Free** plan gives you 2,000 code completions and 50 chat messages per month at no cost — no credit card required.

    If your organization uses **Copilot Business** or **Copilot Enterprise**, ask your admin to assign you a seat.
  </Step>

  <Step title="Install the extension">
    Open VS Code and install the **GitHub Copilot Chat** extension.

    <CodeGroup>
    ```bash Command line
    code --install-extension GitHub.copilot-chat
    ```
    </CodeGroup>

    Or open the Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`), search for **GitHub Copilot Chat**, and click **Install**.

    <Note>
      VS Code 1.115.0 or later is required. The extension version and VS Code version are released in lockstep — always use the latest version of both.
    </Note>
  </Step>

  <Step title="Sign in with GitHub">
    After installation, VS Code displays a prompt to sign in with GitHub. Click **Sign in** and approve the authorization in your browser.

    <Tip>
      If the prompt does not appear, open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) and run **GitHub Copilot: Sign in**.
    </Tip>
  </Step>

  <Step title="Open the chat panel">
    Open the Copilot Chat panel with `Ctrl+Alt+I` (Windows/Linux) or `Cmd+Option+I` (macOS), or click the **Copilot** icon in the Activity Bar.

    The chat panel opens on the right side of VS Code. You'll see a text input at the bottom where you can type your request.
  </Step>

  <Step title="Send your first message">
    Type a prompt in the chat input and press **Enter**. Try one of these examples:

    - `Explain what this file does` (with a file open in the editor)
    - `Write a function that parses a CSV string and returns an array of objects`
    - `What are some best practices for error handling in TypeScript?`
    - `@workspace Find all places where we handle authentication`

    Copilot responds with an explanation, code, or both. For code responses, you can click **Apply** to insert the code directly into your file.
  </Step>

  <Step title="Try inline chat">
    For targeted edits directly in the editor, select some code and press `Ctrl+I` (Windows/Linux) or `Cmd+I` (macOS).

    A small prompt appears inline. Describe the change you want — for example, `Add JSDoc comments to this function` or `Refactor this to use async/await` — and press **Enter**. Copilot shows a diff you can accept or reject.
  </Step>
</Steps>

<Note>
  The **Free** plan includes 2,000 code completions and 50 chat messages per month. If you reach the limit, completions and chat will pause until the next monthly reset. Upgrade to a paid plan for unlimited usage.
</Note>
